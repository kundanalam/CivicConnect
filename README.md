# CivicConnect
🌿 CivicConnect

CivicConnect is a web-based civic complaint management platform designed to help citizens report local issues and enable administrators to monitor, manage, and resolve complaints efficiently.

🚀 Features

- 📝 Register civic complaints
- 📸 Upload complaint images
- 🔍 Track complaint status using Complaint ID
- 📊 Admin Dashboard with live statistics
- 📍 Area-wise complaint monitoring
- 📈 Reports and analytics
- 📥 Export complaints to Excel
- ☁️ Cloud database integration using Supabase
- 🌐 Deployed using Vercel (Frontend) and Render (Backend)

🎯 Objective

The goal of CivicConnect is to provide a simple and transparent platform for reporting and resolving civic issues such as:

- Street Light Problems
- Water Supply Issues
- Garbage Collection Issues
- Road Maintenance Problems
- Other Public Service Complaints
